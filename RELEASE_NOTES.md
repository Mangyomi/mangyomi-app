# Release Notes

## [v1.4.0]

### Features & Improvements
- **Extension Management**: Users can now pick extension priority via drag and drop with persistence.
- **History**: Clicking covers in history now takes you to the manga detail page.
- **NSFW Content**: Implemented NSFW filtering for counts and added "remove from tag" action.
- **Navigation**: Overhauled directory structure for better navigation and resolved minor navigation loops.
- **UI/UX**: various UI improvements and fixes.

### Fixes
- **Stability**: Downgraded Electron to v36.2.0 to address specific issues.
- **Build**: Resolved TypeScript build errors in AniListLinkModal and Sidebar.
- **Visuals**: Fixed extension visual glitches.

### Notes
- This release includes a significant directory restructure.
